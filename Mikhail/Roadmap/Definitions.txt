# Определния из первого блока

AI (Artifical Intelligence) - Искуственный интеленкт предназначенный для решения узконаправленных задач

AGI (Artifical General Inteligence) - Теоритическая попытка создать ИИ который способен самостоятельно обучаться даже тем вещам, 
для которых изначально не предназначался. По русский сильный ИИ.

LLM (Large Language Model) - Большая модель которая обучается на очень больших объемах данных и имитирует человеческую речь. 
Может выполнять практически любую задачу (в текстовом формате), которую можно описать текстом. Примеры: GPT

Training - Обучение. На нем нейросети скармливают датасеты и с помощью них она учиться предсказывать что либо

Inference - "Логический вывод". Это по сути работа ИИ, когда он уже обучен. Когда нейросети скармливают данные, она делает вывод и выдает ответ. 

Embeddings - Процесс/слой в нейросети, который преобразует входные данные (картинки, звуки, слова) в вектора, чтобы нейросеть могла на них обучиться. 

Vector database - База данных с векторами, которые получаются при обучении модели. То есть по сути результат обучения модели, 
который помогает ей быстро работать. ХЗ пока сам не очень понял, разбираюсь

AI Agent - По сути маленькая нейросеть для определенных задач, которая помещается в среду и самостоятельно может принимать решения

RAG (Retrieval-Augmented Generation) - Поиск с использованием нейросети. Это когда вы находите данные по поисковику при этом обрабатывая данные 
с помощью нейроести для более корректного ответа и только после выдаете его пользователю. 

Promt Engineering - Создание "подсказок" для нейросети, которые помогают выдавать ей более точные и желаемые ответы. По сути умение создавать 
для ИИ правильных инструкций.
